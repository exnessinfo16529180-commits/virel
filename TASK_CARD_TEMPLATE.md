### Scope — разрешённые файлы
```
src/components/ui/Button.tsx      ← можно создать/изменить
src/components/ui/Input.tsx       ← можно создать/изменить
src/styles/components.css         ← можно изменить
```
> ⚠️ Работать ТОЛЬКО с файлами из этого списка.

---

### Out of Scope — запрещённые файлы
```
src/store/**          ← НЕ трогать
src/lib/api/**        ← НЕ трогать
src/App.tsx           ← НЕ трогать
src/router/**         ← НЕ трогать
.github/**            ← НЕ трогать
```

### Acceptance Criteria
> Задача считается выполненной, если:

- [ ] `npm run build` проходит без ошибок
- [ ] [Специфичный критерий 1]
- [ ] [Специфичный критерий 2]
- [ ] Компонент работает на мобильном (375px)
- [ ] Нет console.error в runtime

---

### Definition of Done
- [ ] Код написан и работает локально
- [ ] CI Build зелёный в PR
- [ ] Preview URL открывается и выглядит корректно
- [ ] PR description заполнен (что сделано + скриншот/видео)
- [ ] Нет изменений вне Scope
